Projeto Multidisciplinar - Rede "Raízes do Nordeste"

Projeto de software realizado para o projeto multidisciplinar da trilha de Back-End do curso de Análise e Desenvolvimento de Sistemas do Centro Universitário Internacional UNINTER. 

O projeto tem como objetivo desenvolver uma plataforma de restaurante on-line que permita que usuários se cadastrem e façam pedidos, além de permitir que a gestão acompanhe os pedidos e o estoque.
